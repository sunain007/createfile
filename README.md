# createfile

A simple Python script to quickly generate files of any size and type filled with random data.

## Features

- Create files of arbitrary size (in MB)
- Specify any file extension (e.g., pdf, jpg, php, txt)
- Generates files with random byte content

## Requirements

- Python 3.x

## Usage

1. Clone or download the script.
2. Run the script:

```bash
python3 createfile.py
